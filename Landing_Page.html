Summary: Latitude vs. [X]

The purpose of this project is to analyze how certain weather measurements change in regards to Latitdue. This project was accomplished by taking a random sample of 500 cities (at least) from across the world and then obtaining each city's weather data via OpenWeatherMap API.

After all data was obtained, the data was then analyzed using Python and the Import Module MatPlotLib. Many different aspects were analyzed against Lattidue, which included Wind Speed, Humidity, Cloudiness, and Temperature.
